
# 📸 AI Automated Attendance System

This project is a **web-based AI Attendance System** that automatically marks student attendance using **face recognition**. The system detects faces from the camera, matches them with stored student images, and records attendance with date and time in real-time.

🔗 **Live Website:** [https://ai-automated-attendance-system.onrender.com](https://ai-automated-attendance-system.onrender.com)

---

# 🚀 Features

* Real-time face detection using webcam
* Automatic attendance marking (Present / Late / Absent)
* Displays student **name and roll number** after detection
* Attendance stored with **date and time**
* Roll-number-wise structured attendance
* CSV / Excel attendance download
* Admin-only access for data control
* Fast and accurate face recognition
* Fully automated system (no manual attendance)

# 📊 How It Works

1. Student images are preloaded into the system.
2. Camera detects faces in real-time.
3. If matched:

   * Student details are displayed
   * Attendance is marked automatically
4. Attendance is stored with timestamp
5. Data can be downloaded as CSV/Excel
